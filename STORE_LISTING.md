## Permissions Justification

### storage

Used to securely save your Gemini API key and extension settings locally on your device.

### activeTab

Required to read page content and interact with the current tab when you use the chatbot.

### host_permissions: <all_urls>

Necessary because this is a universal website agent that works on ANY website. The extension needs to:

- Read page content to understand context
- Click buttons and fill forms on your behalf
- Inject the chatbot interface on any site you choose

**Important:** The extension only activates when you explicitly use it. It does not run in the background or collect data from websites you visit.
