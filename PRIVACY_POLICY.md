# Privacy Policy for AI Website Agent Chrome Extension

**Last Updated:** February 10, 2026

## Introduction

AI Website Agent ("the Extension") is a Chrome browser extension that helps users interact with websites using AI-powered natural language commands and voice control. This privacy policy explains how we handle your information.

## Information We Collect

### API Key Storage

- **What:** Your Google Gemini API key
- **How:** Stored locally in Chrome's sync storage
- **Why:** Required to communicate with Google's Gemini AI service
- **Access:** Only you have access; stored on your device and Chrome account

### Website Content

- **What:** Text content from websites you visit (when chatbot is active)
- **How:** Read temporarily from the current page
- **Why:** To provide context for AI responses
- **Storage:** Not stored; only sent to Gemini API for processing

### Voice Input (Optional)

- **What:** Voice recordings when using speech-to-text feature
- **How:** Processed by browser's built-in Web Speech API
- **Why:** To convert speech to text for AI commands
- **Storage:** Not stored; processed in real-time only

## How We Use Information

1. **API Key:** Used solely to authenticate requests to Google Gemini API
2. **Page Content:** Sent to Gemini API to generate contextual responses
3. **Voice Input:** Converted to text for AI command processing

## Data Sharing

### Third-Party Services

- **Google Gemini API:** We send your commands and page context to Google's Gemini service. Google's privacy policy applies: https://policies.google.com/privacy
- **No Other Sharing:** We do not share, sell, or transfer your data to any other third parties

### What We DON'T Do

- ❌ We do NOT collect personal information
- ❌ We do NOT track your browsing history
- ❌ We do NOT use analytics or tracking cookies
- ❌ We do NOT sell your data
- ❌ We do NOT store website content permanently

## Data Storage and Security

- **Local Storage Only:** All extension data (API key, settings) is stored locally on your device using Chrome's secure storage API
- **No Remote Servers:** We do not operate any servers or databases
- **Chrome Sync:** If enabled, Chrome may sync your API key across your devices (controlled by Chrome's sync settings)

## Your Rights

You have the right to:

- **Access:** View your stored API key in the extension settings
- **Delete:** Remove your API key anytime by clicking "Clear API Key" in settings
- **Disable:** Turn off the extension or uninstall it completely
- **Control:** Choose which websites to use the extension on

## Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect information from children.

## Permissions Explained

### Why We Need Permissions

1. **`storage`** - To save your API key and settings locally
2. **`activeTab`** - To read page content and interact with websites
3. **`<all_urls>`** - To work on any website you choose to use it on

### How Permissions Are Used

- Permissions are only used when you actively engage with the extension
- The extension does not run in the background collecting data
- You can disable the extension on specific sites
